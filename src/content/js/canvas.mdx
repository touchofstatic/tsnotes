## Canvas is blurry, pixelated, and distorted

Cause: `Canvas.width/Canvas.height` and `CSS width/CSS height` are NOT the same thing.

Canvas dimension is N pixels in the canvas element's bitmap. Default is 300x150px.

Using CSS rules, *the element* is stretched but *the bitmap* stays the same size, distorted to fit the element's size.

Try `const width = (canvas.width = 600);`
