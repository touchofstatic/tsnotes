title: 'db: get the row you insert'
---

aka returning

```
let authorId;
const insertedAuthor = await db
    .insert(Author)
    .values({ name })
    .returning({ insertedId: Author.id });
authorId = insertedAuthor.map((a) => a.insertedId)[0];
```
